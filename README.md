# Мой Трекер — приложение для тренировок Алмата

PWA-трекер тренировок и веса. Владелец: Алмат (цель — похудение, зал, 3 тренировки в неделю).
Сайт: https://56wswpwwbv-beep.github.io/Almat-trakcer/ (установлен на iPhone через «На экран "Домой"»).

## Как устроено

- **Один файл `index.html`** — весь HTML, CSS и JS внутри, без сборки и зависимостей. Так и оставлять.
- `icon.png` (180), `icon-512.png`, `manifest.json` — PWA-иконки и манифест.
- `.github/workflows/pages.yml` — автопубликация на GitHub Pages при каждом push в `main`. Деплой занимает 1–2 минуты.
- Данные пользователя хранятся в `localStorage` (ключ `fittracker_almat`) на телефоне. **Не менять формат state без миграции** — иначе пользователь потеряет данные. Формат: `{profile:{name,height,age,goal}, weights:[{date,kg}], workouts:[{date,plan,exercises:[{name,sets,reps,kg}],note}]}`.
- Резервная копия: кнопка «Отчёт для Claude» кладёт в текст отчёта JSON state; функция `importData()` восстанавливает из него.

## Функции (июль 2026)

- Вкладки: Тренировка / Вес / План / Профиль.
- Запись тренировок (упражнения, подходы, повторы, вес) + «⚡ Заполнить упражнениями дня» по шаблонам `PLANS`.
- Таймер отдыха 60/90/120 сек со звуком (WebAudio) и вибрацией.
- Вес: график динамики (SVG, функция `drawLine`), линия целевого веса, ИМТ.
- Прогресс упражнений: график максимального рабочего веса по датам + рекорд.
- Серии: недели подряд с выполненным планом 3/3, счёт тренировок за неделю.

## Конвенции для Claude

- Язык интерфейса — русский, обращение на «ты», дружелюбный тон, немного эмодзи.
- Дизайн: CSS-переменные вверху файла, светлая + тёмная тема (`prefers-color-scheme`), синий акцент `#2a78d6`/`#3987e5`, системный шрифт.
- Графики: один цвет серии (синий), тонкие линии 2px, точки с обводкой, тултип по касанию, подписи осей приглушённым цветом.
- Перед публикацией тестировать локально (Playwright/Chromium, viewport 390×844, светлая и тёмная тема) и только потом пушить.
- Публикация: `git push` в `main` → GitHub Actions деплоит сам. Для доступа нужен токен владельца (Алмат присылает его в чате; токен в репозиторий не коммитить!).

## Журнал тренировок

Кроме приложения, Claude ведёт журнал наблюдений (вес, тренировки, рекомендации) в чате — Алмат присылает отчёты кнопкой «Отчёт для Claude».
